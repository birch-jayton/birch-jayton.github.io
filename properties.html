❯

Every object has properties associated with it. Each has a name and a value. These values can be hardcoded or dynamically added or changed. Properties are commonly defined with when the objects are created with constructors. We have already used properties in previous examples. Each value associated with our Course object, like the course name, is a property. Let's review this by creating an object with specified properties you can input.
