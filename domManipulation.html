DOM Manipulation

❯

The HTML DOM or Document Object Model is the scheme in which HTML is organized and what JavaScript accesses to manipulate the document. The DOM is organized like a tree with “root” elements that have appending elements off of them or “parent” elements with subsequent “child” elements.

Fetching Elements

One of its most common uses is to allow JavaScript to access elements of the document object, or nodes on the tree. One way this can be done is by assigning an id to an element and calling the document.getElementById() function to get that element directly. We did this in every previous example in which user input needed to be collected from an "input" element.
