Associative Arrays in Local Storage

❯

As mentioned in the earlier lesson about Associative Arrays, true associative arrays don't exist in JavaScript but Objects in JavaScript do use named indexes. Being an object, the Storage objects utilize named indexes. As we have been doing in other examples, when we retrieve data we do it by calling its name. In this example we will nest an Object in our storage Object. The object will be called student and we will call the property called "desiredInternship" to see the student's desired internship.
