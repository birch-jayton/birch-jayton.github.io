Parameters or arguments are the values that a function has to start with to begin their logic. A variable can be passed to a function in the parenthesis following the name of the function. A function with parameters passed to it has already been used in the previous examples. That is the alert() function. The value in the parenthesis is what is alerted to the screen.
In the last example, the addTwo function had logic that consisted of getting the value of the textbox directly from the document. Let’s change it to add two to the number that is passed to it as a parameter and have it return the value.
