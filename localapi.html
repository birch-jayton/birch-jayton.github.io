Local Storage Api

❯

There are many ways to store data with HTML and JavaScript utilizing the users browser. Session Storage holds data for the duration of the session, or until the user clicks away and closes the browser. Local Storage has no expiration date and data persists after the browser is closed. This functionality can be very handy. It can save the shipping information of a customer so they don’t have to enter it in each time they buy something. It can remember if a user is authorized and remember who they are. It can even save the high scores of users of an in browser game. To demonstrate this, the page will ask you for your name. Enter it and click “Remember Me”. Then, most importantly, close your browser and go back to this webpage. Your name will still be there!
