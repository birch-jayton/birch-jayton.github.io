Transitions

Transitions can be cone in just CSS3 without any Javascript needed. This can be triggered by events such as clicking or hovering with the mouse.

Hover over this!

CSS for transition for hover.

Making CSS transitions is easy. You first make a pseudo-class to define the style of what you want for the final state of the transition. In the above example we created pseudo-class for when the user hovers over a div that is of the "hoverExample" class by adding ":hover". The "transition" property allows us to define how long we want the transition to take. We are able to specify different durations for each property of the style. In this case, the change in width will last 2 seconds and the background color change will last 3. (Line 4). The transition timing function controls the speed curve of the transition and is set to "ease" by default. Tis means its not the same speed the entire transition but "eases" in and out of he transition looking a little smoother.

Animations





Similar to transitions, animations allow you to change CSS properties. The @keyframes rule defines the animation. You must then bind to the class that you want to give the animation to.
