Variables

❯

A Variable is memory that contains a data value of a certain data type. It is a piece of information that the programmer can use to perform certain tasks. For example, when you perform the task of saying your friend’s name when you greet them, the name used is a variable in your memory that you use to perform the “function” of saying your friend’s name.

A Data Type is the classification of data so the compiler knows what the programer wants the data to represent.

The primitive or basic data types in JavaScript are:

  • Boolean
  • Null
  • Undefined
  • Number
  • String
  • Symbol

Using the friendly greeting as an example, your friend’s name is obviously a string of English characters, not a number. The programmer must use the correct data types for the respective variables that they create. If they use a floating point variable that is 32-bits long to record whether something is a 0 or 1 then that would be a big waste of memory (like using a quarter of your long term memory to remember your friends name). Luckily, this is easy in JavaScript. Using the keyword var will declare your variable and and also detect the proper datatype as determined by the value you assign it.
