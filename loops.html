Loops

❯

Loops are used to go through the same code over and over or “loop” through it. The types of loops in JavaScript are:

  • for - loops through a block of code a number of times.
  • for/in - loops through the properties of an object.
  • while - loops through a block of code while a specified condition is true.
  • do/while - also loops through a block of code while a specified condition is true.

Lets demonstrate this loops using the example object (class data) and array (animal list) from the last example.

To demonstrate the FOR loop we will iterate through the list of animals and whenever we come across a dog we will increment the dog_count by one. It will then display the dog count.

To demonstrate the for-in loop we will iterate through each value of the cit_class object and append it to one string then display it.

To demonstrate the while loop we will start with an integer at 0. We will add the value of that number to a string then implement it by one. This will keep appending to the string until the integer reaches 10. NOTE: This will be displayed in the console. To see the reult use the developer tools of your browser.
