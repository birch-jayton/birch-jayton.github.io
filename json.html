JSON

❯

JavaScript Object Notation or JSON is text that adheres to a syntax that is used to describe objects. We cannot send JavaScript object from the browser to the server. We can only send text. The text we send when we want to do this is JSON. We have already looked at JavaScript objects in a few examples, onew being the Course object we use to describe a class at school. As described in the Associative Arrays lesson, JavaScript objects use a named index. This means if we want to get the name of our class being represented as a JavaScript object called cit_class then we would just need to call “cit_class.name”. This exists in our JavaScript environment as a JavaScript object. The browser has our .js file and interprets it as an object. But what do we do if we want to send this data to a different server? We turn the object into plain text being JSON and we let the other server decide how to interpret it. The JSON looks very similar to how we wrote the object on line 2.
