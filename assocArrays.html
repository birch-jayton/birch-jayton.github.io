Associative Arrays

❯

An associative array is where a variable in an array has a named index. Sometimes referred to as dictionaries.
JavaScript does NOT allow associative arrays. All arrays in JavaScript have a numbered index. Objects in JavaScript however do used a named index. This comes in handy when retrieving using JSON objects.

Notice the distinction between how the values in a numbered index (line 13) are retrieved compared to a named index (line 18). One is called with a number (animalsInYard[0]) and one by the name assigned to the value (cit_class.name)
