Stringify/Parse

❯

In order to convert a JavaScript object to a plain string of JSON we use the JSON.stringify() function. This string can then be used to send to another server so that the server can interpret it as an Object. Let’s demonstrate this by taking our cit_class JavaScript object that we have been using and displaying it as a string of text in an alert.

We can use that string to send it somewhere. So what is a developer supposed to do when receiving a string like this? We can convert the JSON to be a regular JavaScript Object by having the interpreter parse the text and create the object. This is done with the JSON.parse() function. We did this previously in our Using XMLHTTPRequest lesson when we got JSON back from the database. In order to use this information as an object we parsed it and put the data into a table.
