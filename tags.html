HTML 5 Tags

❯

HTML 5 provides tags that make things possible that required browsers with plugins like flash in the past. This includes the ability to embed audio and video. These tags act like other basic HTML tags with properties that you can modify to suit the embeddings to your needs.

The Video Tag

Here is a video of my dog, Jango, sneezing.

Pretty gross right? I think the face he makes is funny. Anyways... similar to the img tag, you specify the size of the element as well as the source file. It is always good practice to add some alternative text in case the file cannot load for some reason. One cool thing about the HTML 5 video tag is the "controls" attribute that lets you define youtube-like controls for your video to play/pause, control volume, full-screen, etc.

The Audio Tag

The audio tag is almost identical to the video tag but plays audio files instead. You specify controls and the source file.

The Canvas Tag

This tag is used to create a canvas on which you can dynamically create graphics with Javascript
