Javascript Events

Almost all of the examples in previous lessons have executed Javascript functions. The scripts run under the condition that a certain event happens. The most common one used was onclick, meaning that the function runs when the user clicks something.

Common events include:

  • onchange
  • onclick
  • onmouseover
  • onmouseout
  • onkeydown
  • onload

Mobile Specific Events

HTML 5 has specific DOM events for mobile devices called Touch Events. Similar to a mouse clicking this involves when the user touches their finger to the screen.

These touch events are:

  • ontouchcancel
  • ontouchend
  • ontouchmove
  • ontouchstart

To test this we will have an element trigger an event when the user places their finger on the screen and moves it.
