Conditional Statements

❯

Condition Statements are used to perform certain actions depending on certain conditions. In JavaScript there are the following conditional statements:

  • Use if to specify a block of code to be executed, if a specified condition is true.
  • Use else to specify a block of code to be executed, if the same condition is false.
  • Use else if to specify a new condition to test, if the first condition is false.
  • Use switch to specify many alternative blocks of code to be executed.

Let's ask for the user to input a number and we'll suppose that 7 is my FAVORITE number. So if the number is 7 we will act a certain way. However, 3 is a close second when it comes to by favorite numbers so we will also have a behavior on the condition that the number is not 7 but 3.
