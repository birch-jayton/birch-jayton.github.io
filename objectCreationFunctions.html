In JavaScript objects are everywhere. As in our example of our cit_class function, objects hold values and methods that can be retrieved by calling their name. Let's say that you are making a program in which you will need various objects to represent classes like our cit_class object. Instead of creating each of these objects literally you can create a constructor to make these objects. Constructors are functions in which an object is created based on the values passed as parameters. Let's create a constructor so that we can make more objects to represent our other classes. Notice that when the constuctor is called the keyword new is used to indicate that a new object needs to be created to assign to the variable name. Using a constuctor is handy because all of the objects created with it will have the same member variable names. So if we create 100 Course objects we know we can always call course.name to get the name.
