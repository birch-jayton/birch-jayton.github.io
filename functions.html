Functions

❯

Now that we can save pieces of data as variables we can use that data to give a program functionality and make it do computations or change the data. This is done with functions. A function or method is a chunk of code designed to perform a certain task with data in the program. JavaScript uses the "function" keyword followed by the name of the function. Parameters are placed in parentheses and the logic of the function is placed in curly brackets. A return value can be used to send the returned value to the caller.

Example

Let's create a program that adds two to any number entered by the user. Because functions do things, it is good naming convention to give them verb phrases as names to reflect what the function does. For our function that adds two we will call it "addTwo". After the arithmetic of add 2 to a number the function will “return” the solution to the caller.
